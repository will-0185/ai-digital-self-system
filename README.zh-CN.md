# AI 数字分身系统

[English](./README.md) | **中文**

一套本地优先的文件系统框架，用于构建、维护和演化个人 AI 数字分身。

它试图长期保存和调用一个人的：

- 思考方式
- 判断方式
- 价值排序
- 创作方式
- 长期变化

这个项目不是聊天机器人人格，不是单个 Prompt，也不是简单的记忆增强方案。它是一套面向长期自我建模的 AI 协作系统。

## 这是什么？

大多数 AI 对话都是临时的。上下文会消失，记忆不稳定，一个人更深层的判断方式也很难被长期保存。

AI 数字分身系统用本地文件系统作为权威源，解决这个问题。

它把以下内容分开：

- 系统演化
- 内容创作
- 个人素材
- 知识资源
- 周期性自我审查

最终形成一套可迁移、可迭代、与模型无关的个人 AI 系统框架。

## 核心架构

```text
00_System_Overview      系统说明与配套提示词
01_System_Kernel        元认知、元知识、核心人格
02_Knowledge_Creation   外部资料、skills、文章、知识笔记
03_Material_Record      月度原始素材记录
04_Evolution            月度、季度、年度演化审查
```

## 快速开始

1. 阅读 `00_System_Overview/AI_Digital_Self_System.zh-CN.md`
2. 将系统结构复制到你的本地文件系统
3. 填写自己的系统内核
4. 按月记录素材
5. 按月度、季度、年度进行演化审查
6. 使用配套提示词在你偏好的 AI 模型中运行系统

## 公开版与私人版边界

本仓库是公开框架和公开示例，不是作者的完整私人数字分身。

**不要把自己的真实数字分身系统直接放进公开仓库。**

以下内容应保留在私人系统或私有仓库中：

- 真实月总结
- 私人关系
- 生活流水
- 证据索引
- 未发布作品
- 敏感个人材料
- API key、token、账号凭证

详见 `00_System_Overview/Privacy_Boundary.zh-CN.md`。
