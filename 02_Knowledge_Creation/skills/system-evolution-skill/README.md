# System Evolution Skill

A structured workflow for running periodic reviews of your AI Digital Self System.

## What this skill covers

- Monthly trend analysis
- Quarterly personality review
- Annual evolution report
- Kernel file revision workflow

## How to use

1. Copy the **SKILL.md** prompt into a new AI conversation
2. Upload the required material files
3. Follow the prompt instructions to generate the review
4. Review the output and decide on revisions
5. Archive old versions before making changes
