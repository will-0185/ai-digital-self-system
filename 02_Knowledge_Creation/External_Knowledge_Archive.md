# External Knowledge Archive

A single-file archive of external knowledge, organized by topic. Each entry includes links, excerpts, and AI summaries.

## Format

```markdown
### Topic

- Source: [title](url)
- Excerpt: ...
- Summary: ...
- Paradigm candidate: ...
```

## Example entries

### AI & Memory Systems

- Source: [Article about memory systems in AI](https://example.com/memory-systems)
- Excerpt: "A truly stable intelligent system cannot rely solely on long context windows to 'stuff' information in. It must possess continuous learning, memory integration, long-term planning, and consistency."
- Summary: This reframes the AI memory problem from "context window size" to "memory architecture design." Relevant to digital self system design.
- Paradigm candidate: Stable intelligence requires memory integration, long-term planning, and consistency — not just larger context windows.

### Knowledge Management

- Source: [Article about personal knowledge systems](https://example.com/pkm)
- Excerpt: "The most effective knowledge systems don't just store information — they continuously compile raw materials into structured, traceable, updateable knowledge assets."
- Summary: Shifts the focus from collection to compilation. Influences how the digital self's knowledge base should be structured.
- Paradigm candidate: Knowledge layering and continuous compilation as a core paradigm.

### Social & Political Analysis

- Source: [Analysis of organizational capacity](https://example.com/org-capacity)
- Excerpt: "Resource wealth does not equal strategic autonomy. Resources must be converted into organizational, military, institutional, and decision-making capacity to constitute real power."
- Summary: A useful lens for analyzing why some resource-rich entities lack strategic influence. Applicable beyond geopolitics to organizational behavior.
- Paradigm candidate: Resource wealth ≠ strategic autonomy; conversion capacity matters.

## Note

This is a public-safe template with example entries. Real entries link to actual sources and contain the user's own summaries and paradigm extractions. Do not include full copyrighted materials — keep excerpts short and link to originals.
