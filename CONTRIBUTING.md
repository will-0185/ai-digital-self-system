# Contributing

Contributions are welcome! This project aims to build a general framework for AI digital self systems.

## How to contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-idea`)
3. Commit your changes
4. Open a Pull Request

## Guidelines

- Keep the architecture clean and minimal
- Do not include private or personal data in examples
- Maintain dual-language support where applicable
- Respect the existing directory structure
