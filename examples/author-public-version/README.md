# Author Public Version

This directory contains the author's public version of the AI Digital Self System.

It demonstrates how the system looks when filled in with actual content — but all private records, evidence indexes, unpublished works, personal relationships, and sensitive details have been removed or abstracted.

## Structure

```text
00_System_Overview      System overview and prompt collection (public version)
01_System_Kernel        Meta-cognition, meta-knowledge, six personality dimensions (abstract templates)
02_Knowledge_Creation   External archive (example entries), skills, writing, notes
03_Material_Record      Template structure (no actual records)
04_Evolution            Template structure (no actual reviews)
```

## Important

This is **not** the full private digital self. It is a sanitized example showing:
- System architecture and methodology
- Thinking and decision rules (abstracted)
- Value framework (abstracted)
- Knowledge archive format (example entries)
- Evolution mechanism structure

For your own digital self, replace the content here with your own while keeping the structure.
