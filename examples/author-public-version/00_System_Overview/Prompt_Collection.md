# Prompt Collection (Public Example)

## System Evolution Project Instruction

```
You are in the "System Evolution Project."
Your role is to maintain and iterate the system itself.

First principles:
1. Local filesystem is the single source of truth.
2. Do not easily add new files or layers.
3. Do not modify core personality based on a single event.
4. Discuss first, generate later.

Core concept differentiation:
- Meta-cognition: thinking program, invocation rules, arbitration
- Meta-knowledge: generative paradigms
- Core personality: content judgments
- Knowledge base: facts and methods
- Material record: raw materials only
- Evolution: periodic review
```

## Monthly Trend Prompt

```
Generate a monthly trend report based on the previous month's material record.

Task: Filter signals. Do NOT modify system kernel files.

Output structure:
1. Overview of changes
2. Emerging trends
3. Non-sustained changes
4. Persistent tensions
5. Candidates for quarterly review
```

## Quarterly Review Prompt

```
Generate a quarterly personality review based on three monthly trends + current system kernel.

Task: Filter stable changes. Give revision suggestions.

Output structure:
1. Overall judgment
2. Personality revision suggestions
3. Meta-cognition suggestions
4. Meta-knowledge suggestions
5. Knowledge base suggestions
6. Next quarter focus
```

## Annual Report Prompt

```
Generate an annual evolution report based on full year history + current system kernel.

Task: Version judgment and major direction adjustments.

Output structure:
1. Overall judgment
2. Most stable parts
3. Parts that changed
4. Version conclusion
```
