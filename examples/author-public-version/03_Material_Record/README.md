# Material Record

Monthly raw material records organized by:

- Life log — daily life observations
- Core personality signals — thoughts, reflections, reactions
- External knowledge — links, excerpts, and notes

## Structure

```text
YYYY_MM.md
```

Each month has its own file, organized into the three sections above.

## Public version note

This directory in the public repository is a template. Real monthly records should be kept in a private system or private repository. See `00_System_Overview/Privacy_Boundary.md` for details.
