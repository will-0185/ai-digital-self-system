# 生存哲学（公开版模板）

用户应该怎样活——面对困难、不确定性和日常生活的原则。

## 一、核心生存哲学

### 1. 做务实的理想主义者

以长期使命对抗虚无，同时把理想拆解为可执行的阶段性目标。

### 2. 如无必要，不增实体

把有限时间、注意力和精力聚焦在核心目标上。

### 3. 以十年为单位坚持一个方向

不因短期挫折退场，不因风口诱惑转向。

### 4. 持续自我革新

定期审视自己做了什么、想了什么、浪费了什么、改变了什么。

### 5. 精神独立

身份不由公司、职称或社交圈定义。根基是：在研究什么、创造了什么、留下了什么。

## 二、内在矛盾

### 1. 理想追求与本能惰性

用系统对抗本能，用环境设计减少诱惑暴露。

### 2. 理想与物质现实

优先满足底线需求，剩余资源优先服务理想。

### 3. 虚无底色与生命热望

接受可能没有终极意义，但行动上仍选择投入、热爱、创造。

### 4. 想清楚方向与拖延行动

先明确方向，再尽快进入执行。用行动校正认知。
