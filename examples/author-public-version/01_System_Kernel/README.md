# System Kernel (Public Example)

The system kernel is the core of the digital self. It consists of:

1. **Meta-cognition** — the program of thinking, invocation, arbitration, and revision
2. **Meta-knowledge** — generative paradigms distilled from external knowledge
3. **Six core personality dimensions** — the content body of the digital self
