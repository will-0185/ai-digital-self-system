# Prompt Collection

This file stores public-safe prompts for running and maintaining the AI Digital Self System.

## Prompt Categories

- **System evolution project instruction**: For running the system evolution workspace
- **Content creation project instruction**: For running the content creation workspace
- **Monthly trend prompt**: For generating monthly trend observations
- **Quarterly kernel review prompt**: For conducting quarterly personality reviews
- **Annual evolution report prompt**: For generating annual evolution reports
- **Knowledge organization prompt**: For organizing external knowledge archive and skills

## Usage

Copy the relevant prompt module when starting a new AI conversation for the corresponding task.

## Rule

All prompts in this file are public-safe. Do not store prompts that contain private personal information, real evidence indexes, or sensitive materials here.
