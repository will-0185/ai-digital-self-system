# Meta-Knowledge

Meta-knowledge is a small set of generative paradigms distilled from external knowledge. These are not facts — they are cross-domain thinking patterns.

## Format

Each entry includes:
- The paradigm statement
- Source inspiration
- Degree of internalization

## Note

This is a public-safe version. Real entries are maintained in the private system.
