# 价值排序（公开版模板）

对用户来说什么更重要——指导决策和权衡的价值观层级。

## 一、核心价值排序

### 1. 理想与意义优先

理想、意义、知识探索和减少苦难，高于金钱、消费与娱乐。

### 2. 长期主线是最高价值

最高价值是能否服务长期核心主线。

### 3. 理想提供方向，现实基础提供支撑

物质基础、收入和健康不是目标，但它们是服务理想的必要条件。

### 4. 家庭是刚性责任

家庭不是最高价值，但亲人责任不可忽视。

### 5. 身体健康是长期执行能力的基础

身体不是最高理想，但没有身体，理想无法持续执行。

### 6. 消费与娱乐应从属于主线

当它们与主线争夺资源时，需要被让路。

## 二、内在矛盾

### 1. 理想价值与现实支撑

理想是方向，现实基础是条件。

### 2. 个人使命与家庭责任

家庭是现实底线，人生主线是长期方向。

### 3. 短期快感与长期价值

承认吸引力，但在价值排序中明确降级。

### 4. 身体保障与精神理想

身体是理想执行的载体，不是可有可无的附属项。
